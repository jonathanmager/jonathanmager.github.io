Portfolio

Braves Regression Project

This project was a group project with 4 other grad students exploring (descriptive analysis) what MLB box scores contributed to run differential for the Atlanta Braves. My main contribution was exploring how lasso regression could be used on this problem. The reason I explored lasso regression was because Lasso that uses a penalty term to shrink coefficients to avoid overfitting AND has the ability to set some coefficients to 0. This is useful because it can do automatic dimensionality reduction for variables that are not as useful. My lasso model was the best performing method in the group on "predicting" run differential for the 2023 season so far.
